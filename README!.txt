Welcome to the 2D Normal Mapping Tutorial!
Prerequesites: Sprite Batch, Normal Mapping (3D)

In this tutorail we show a neat illusion where you can 
use normal mapping on 2D surfaces, to give the illusion of
3D detail. After learning how to use 3D graphics in the 
"More Graphics" section, this tutorial should be relatively easy
